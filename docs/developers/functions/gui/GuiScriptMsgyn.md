# GuiScriptMsgyn

Pop up a message box containing yes/no choices.

```c++
int GuiScriptMsgyn(const char* message);
```

## Parameters

`message` The content of the yes/no choices message box.

## Return Value

1 if the user chose yes, 0 if the user chose no.
