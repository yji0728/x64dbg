# GuiCloseQWidgetTab

Close a Qt QWidget tab added by a plugin.

```c++
void GuiCloseQWidgetTab(QWidget* qWidget);
```

## Parameters

`qWidget` The Qt QWidget object of the tab to close.

## Return Value

This function does not return a value.
